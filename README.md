# node-cache-store

A in-memory cache system.

* Key/Value store.
* You can use multi cache space like namespace.

# Installation

`npm install node-cache-store`

# Usage

```javascript
var Cache = require('node-cache-store').Cache;
var cache = new Cache() ;

// Set a cache
cache.set("key", "value");

```
